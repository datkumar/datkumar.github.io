## Routes

```js
GET      /api/products
GET      /api/products/:id
POST     /api/products
PUT      /api/products/:id
DELETE   /api/products/:id
```

## Instructions

```sh
# Install dependencies
npm install
# Run project
npm run dev   # in development mode
npm start     # in production mode
```
